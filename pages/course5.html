Full stack development - Python Django

Course Description

Master the art of building complete web applications using Python and its robust Django framework. Learn front-end technologies like HTML, CSS, and JavaScript alongside back-end development with Python and Django, gaining a comprehensive understanding of the entire web development process.

Course Modules

Module 1: Core Python Programming

Learn the fundamentals of Python programming, including data types, control flow, functions, and object-oriented programming.

Module 2: Django Framework

Understand the Model-View-Controller (MVC) architecture and its implementation in Django.

Module 3: Web Development Fundamentals

Learn HTML, CSS, and JavaScript for building the front end of web applications.

Module 4: Database Design and Management

Learn to design and manage databases using Django's ORM or other database tools.

Module 5: API Development

Build RESTful APIs to interact with your Django applications and other services.
