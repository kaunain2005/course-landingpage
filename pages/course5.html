Full Stack Development - Backend Python Django

Course Objective

The objective of this course is to equip students with the necessary skills to design, develop, and deploy Full Stack Web Applications using modern technologies like Django, Bootstrap, and MySQL. Students will learn to implement best practices in backend development, database management, and frontend design, while gaining experience in version control through Git. The course also aims to teach the fundamentals of Test Driven Development (TDD) to ensure high-quality code and robust applications. Finally, students will be introduced to the principles of continuous integration and deployment, enabling them to confidently deploy applications to cloud platforms.

Course Modules

Module 1: Introduction to Full Stack Development (6 Hours)

This module introduces full stack development concepts using technologies like Python Django, MySQL, Bootstrap, Git, and testing practices.

    1: Overview of Full Stack Development (1.5 Hours)
  • Concepts Covered:
    • Introduction to full stack development
    • Key technologies: Backend (Django), Frontend (Bootstrap), Database (MySQL), Version Control (GIT), and Testing (TDD)
  • Practical Activity: Set up a simple full-stack development environment with Django, Bootstrap, and MySQL.
    • 2: Setting Up the Environment (1.5 Hours)
  • Concepts Covered:
    • Installing Python, Django, MySQL, Git
    • Setting up a virtual environment
    • Installing necessary libraries
  • Practical Activity: Install and configure Python, Django, MySQL, and Git on your system.
    • 3: Version Control with GIT (3 Hours)
  • Concepts Covered:
    • Introduction to Git and GitHub/Bitbucket
    • Basic Git commands (init, clone, commit, push, pull, and branch)
    • Setting up a project repository and pushing initial code
  • Practical Activity: Create a GitHub repository and push your first Django project.

Module 2: Backend Development with Python Django (12 Hours)

This module covers backend development with Django, including project setup, working with models, and building APIs.

    1: Django Project Setup (3 Hours)
  • Concepts Covered:
    • Creating a new Django project
    • Overview of Django MVC (Model, View, Controller)
    • URL routing and views
  • Practical Activity: Create a new Django project and set up basic views and URL routing.
    • 2: Working with Models and Migrations (3 Hours)
  • Concepts Covered:
    • Defining models in Django
    • Running migrations to create database tables
    • Setting up relationships between models
  • Practical Activity: Define models and run migrations in Django to create database tables.
    • 3: Building APIs with Django REST Framework (3 Hours)
  • Concepts Covered:
    • Setting up Django REST framework
    • Creating basic CRUD operations using Django views and serializers
    • Understanding API routing and response handling
  • Practical Activity: Build a simple API with Django REST framework.
    • 4: Authentication and Permissions (3 Hours)
  • Concepts Covered:
    • Implementing user authentication (JWT, session-based auth)
    • Setting up permissions and restricting access to APIs
    • Using Django's built-in user authentication system
  • Practical Activity: Implement user authentication in your Django API.

Module 3: Frontend Development with Bootstrap (10 Hours)

This module covers frontend development using Bootstrap and how to integrate it with Django for dynamic web applications.

    1: Introduction to Bootstrap (3 Hours)
  • Concepts Covered:
    • Overview of Bootstrap grid system and responsive design
    • Setting up a simple Bootstrap template
    • Customizing Bootstrap components (buttons, forms, navbars, etc.)
  • Practical Activity: Set up a basic Bootstrap template in your Django project.
    • 2: Creating Frontend for Django APIs (3 Hours)
  • Concepts Covered:
    • Integrating Bootstrap with Django templates
    • Designing user interfaces for API data display
    • Using Bootstrap forms to submit data to Django backend
  • Practical Activity: Design a frontend for Django APIs using Bootstrap.
    • 3: Advanced Bootstrap Features (4 Hours)
  • Concepts Covered:
    • Building more complex layouts using Bootstrap
    • Adding interactivity with Bootstrap modals, carousels, tooltips
    • Responsiveness: media queries, breakpoints, and customization
  • Practical Activity: Enhance your Django frontend with advanced Bootstrap features.

Module 4: Database Design and Integration with MySQL (10 Hours)

This module covers MySQL database integration with Django, including database design and ORM usage.

    1: Introduction to Relational Databases and MySQL (2 Hours)
  • Concepts Covered:
    • Overview of relational databases and SQL
    • Setting up MySQL database and connecting it with Django
  • Practical Activity: Set up MySQL and connect it with your Django project.
    • 2: Database Design and Schema Creation (3 Hours)
  • Concepts Covered:
    • Designing efficient database schemas
    • Understanding relationships (one-to-many, many-to-many)
    • Database normalization principles
  • Practical Activity: Design a database schema and create tables in MySQL.
    • 3: Working with Django ORM and MySQL (3 Hours)
  • Concepts Covered:
    • Using Django's ORM to interact with the MySQL database
    • Running complex queries, filtering, and aggregating data
    • Optimizing queries and managing large datasets
  • Practical Activity: Use Django ORM to query MySQL data.
    • 4: Database Migrations and Data Seeding (2 Hours)
  • Concepts Covered:
    • Handling database migrations and schema updates
    • Data seeding: adding initial data to the database
  • Practical Activity: Add seed data to your database using Django migrations.

Module 5: Deployment with GIT and Cloud (6 Hours)

This module covers deployment strategies for Django applications using Git and cloud platforms like Heroku or AWS.

    1: Version Control Best Practices (2 Hours)
  • Concepts Covered:
    • Working with branches and merging
    • Pull requests and code reviews
    • Tagging and release management
  • Practical Activity: Use Git for version control and collaboration in a team project.
    • 2: Deploying Django Application on Cloud (Heroku/AWS) (4 Hours)
  • Concepts Covered:
    • Setting up a Heroku/AWS account and deploying the application
    • Configuring environment variables and scaling the application
    • Setting up continuous integration/continuous deployment (CI/CD) pipelines with GitHub Actions or CircleCI
  • Practical Activity: Deploy your Django application on Heroku or AWS and manage deployments.

Module 6: Test Driven Development (TDD) and API Testing (4 Hours)

This module focuses on Test Driven Development (TDD) principles and API testing using Django.

    1: Introduction to TDD in Django (2 Hours)
  • Concepts Covered:
    • Writing test cases for Django models, views, and serializers
    • Understanding the TDD process (Red, Green, Refactor)
    • Setting up testing tools (pytest, unittest)
  • Practical Activity: Write and run test cases for your Django models, views, and serializers using pytest or unittest.
    • 2: API Testing with TDD (2 Hours)
  • Concepts Covered:
    • Writing test cases for REST API endpoints
    • Running tests and interpreting results
    • Automating tests with CI tools (GitHub Actions, Jenkins)
  • Practical Activity: Implement and run tests for your Django REST API endpoints and automate testing with CI tools like GitHub Actions.

Module 7: Final Project and Review (6 Hours)

This module provides an opportunity to work on a full-stack project to consolidate the knowledge gained throughout the course.

    1: Building a Full Stack Project (6 Hours)
  • Concepts Covered:
    • Backend: Set up Django models and APIs
    • Frontend: Design UI with Bootstrap and integrate it with APIs
    • Database: Implement MySQL for data storage and queries
    • Deployment: Deploy application to the cloud using Git-based CI/CD
    • Testing: Implement TDD, write test cases for API endpoints
  • Practical Activity: Build a full-stack application by integrating Django backend with a Bootstrap frontend, using MySQL for database storage, deploying to the cloud (Heroku or AWS), and automating the testing process using TDD and CI/CD tools.

Course Outcome

Students will be able to develop and deploy fully functional full stack web applications using Django for backend development and Bootstrap for frontend design. They will have proficiency in managing databases with MySQL and effectively using Git for version control and collaboration. Students will also be able to implement Test Driven Development (TDD) practices to ensure the reliability and quality of their code. Additionally, they will gain hands-on experience in deploying applications to cloud platforms and setting up CI/CD pipelines for continuous integration and delivery.
